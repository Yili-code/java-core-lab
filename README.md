# java-core-lab

放一些 Java 基礎跟 OOP 的小範例、練習題，純個人練習用。

## 目錄大概長這樣

```text
lab/javacore/
├── oop/
├── variable/
├── datatype/
├── switchdemo/
├── operator/
├── defaultt/
└── practice/      (p001～p012)

note/              
```

## 編譯與執行

```bash
javac lab/javacore/oop/*.java
java lab.javacore.oop.test
```
